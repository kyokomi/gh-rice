gh-rice
============================
A github cli tool only to comments for golang

## Install

```bash
go get github.com/kyokomi/gh-rice
```

## Usage

```bash
$ gh-rice -h
Usage of ./gh-rice:
  -b string
    	github branch name
  -c string
    	comment
  -o string
    	github owner name
  -r string
    	github repository name
  -t string
    	github access token
```
